q4.calendar

An interactive calendar with links to events.

  • Options

Author(s)

Requires
  • Underscore.js
  • Moment.js
  • CLNDR.js

Examples

$("#clndr").calendar({
    news: true,
    presentations: true,
    eventSize: 50,
    slideshare: "Q4WebSystems"
});

Options

  • publicFeed

    boolean

    This allows the widget to be placed on a site not hosted by Q4. Requires url and apiKey to be set in the configuration.

    Default value
    false

  • url

    string

    A URL to a Q4 hosted website. This is only required if publicFeed is true.

  • apiKey

    string

    The API Key can be found under System Admin > Site List > Public Site in the admin of any Q4 website. This is only required if publicFeed is true.

  • news

    boolean

    Whether to include all related press releases.

    Default value
    false

  • presentations

    boolean

    Whether to include all related presentations.

    Default value
    false

  • slideshare

    string

    Can be set to a SlideShare username. This will add SlideShare presentations as events.

  • onEventsLoad

    function

    A callback fired after events are loaded.

  • onSlideShareLoad

    function

    A callback fired after SlideShare has loaded.

  • eventSize

    number

    The maximum number of events to add to the calendar.

    Default value
    25

  • tags

    Array<string>

    An array of tags used to filter events.

  • calendar

    An options object to pass directly to the CLNDR.js instance. See that module's documentation for details.

    Example
    {
    adjacentDaysChangeMonth: true,
    daysOfTheWeek: ['Su','Mo','Tu','We','Th','Fi','Sa'],
    showAdjacentMonths: false,
    weekOffset: 1,
    doneRendering: function () {
        console.log('done rendering')
    },
    ready: function () {
        console.log('calendar is ready')
    }
}
