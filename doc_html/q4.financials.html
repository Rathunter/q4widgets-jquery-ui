q4.financials

Fetches and displays financial reports from the Q4 private API.

Source file
q4.api.js, line 710
Script link
q4.api.1.1.0.min.js
Extends
  • q4.api - see this documentation page for more options and methods.

Options

  • reportTypes

    Array<string>

    A list of report subtypes to display, or an empty list to display all. Valid values are:

    • Annual Report
    • Supplemental Report
    • First Quarter
    • Second Quarter
    • Third Quarter
    • Fourth Quarter

  • docCategories

    Array<string>

    A list of document categories to display. Use an empty list to display all.

    Example
    ["Financial Report", "MD&A", "Earnings Press Release"]

  • shortTypes

    Object

    A map of short names for each report subtype, for use in the template.
